# Exercícios Resolvidos

Esta seção documenta os exercícios práticos desenvolvidos para fixação dos conceitos. Cada arquivo representa um problema resolvido.

## Lista de Exercícios

| Arquivo | Descrição / Conceito Praticado |
| :--- | :--- |
| `1ExSentence.kt` | Manipulação básica de Strings e Output. |
| `2ExRound.kt` | Arredondamento e conversão de tipos (Double para Int). |
| `3ExMixOperators.kt` | Precedência de operadores aritméticos e lógicos. |
| `4ExMixOperators2.kt` | Mais prática com expressões complexas booleanas. |
| `5ExMaxValue.kt` | Lógica de decisão (`if/else`) para encontrar maior valor e limitar ranges. |
| `6ExParImpar.kt` | Uso do operador módulo `%` para verificar paridade. |
| `7ExAgeGroup.kt` | Classificação de faixas etárias usando múltiplos `if/else`. |
| `8ExCalculator.kt` | Calculadora simples usando `when` e entrada de dados. |
| `9ExTabuada.kt` | Estrutura `while` para gerar tabuada. |
| `9ExTabuadaFor.kt` | Estrutura `for` (Ranges) para gerar tabuada. |
| `10ExGuessNumber.kt` | Jogo de adivinhação usando `Random` e loops `while`. |
| `11ExMask.kt` | Manipulação de Strings (máscara de senha) e tratamento de Nulos (`?.`). |
| `12ExCalc.kt` | Encadeamento de chamadas (method chaining) e `toIntOrNull`. |
| `13ExDogAge.kt` | Função simples com retorno (Idade de cachorro). |
| `14ExStringSpacing.kt` | Função para manipular strings (espaçamento entre letras). |
| `15ExRandom.kt` | Geração de caracteres aleatórios e classes utilitárias simples. |
| `16EXDrinkingWather.kt` | Exercício completo de POO: Classe `Person`, métodos de estado (`walk`, `drinkWater`), propriedades. |
