# Main.kt

Este arquivo contém o exemplo clássico de "Hello World" em Kotlin.

## Conceitos
- **Função `main`**: O ponto de entrada de uma aplicação Kotlin.
- **`println`**: Função padrão para imprimir texto no console com quebra de linha.
- **Sintaxe simplificada**: Diferente de Java, não é necessário uma classe invólucro para a função main.

```kotlin
fun main() {
    println("Hello Kotlin")
}
```
