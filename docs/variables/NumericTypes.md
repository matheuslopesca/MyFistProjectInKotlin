# Tipos Numéricos (variables2Numbers.kt, variables3.kt)

Kotlin possui tipos ricos para números inteiros e flutuantes.

## Conceitos
- **Inteiros**: `Byte`, `Short`, `Int`, `Long`.
- **Ponto Flutuante**: `Float`, `Double`.
- **Sufixos Literal**:
  - `L` para Long (ex: `100L`).
  - `F` para Float (ex: `10.5F`).
- O padrão para inteiros é `Int` e para decimais é `Double`.

# Declaração Explícita
Ao declarar `val i2: Int = 10`, estamos dizendo explicitamente o tipo, perdendo a brevidade da inferência mas ganhando clareza se necessário.
